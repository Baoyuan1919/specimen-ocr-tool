# 图片OCR识别 → Excel自动填表工具

从文件夹批量读取图片 → OCR识别文字 → 按字段名提取内容 → 自动填入Excel

## 功能特点

- **批量处理**：一次处理文件夹内所有图片（jpg/png/bmp/tiff/webp）
- **中文优化**：基于PaddleOCR，中文识别精度高
- **智能字段匹配**：自动识别 "字段名：值" 格式的内容
- **自动样例提取**：扫描一张图片，自动识别其中文本供你勾选字段
- **已有Excel导入**：直接读取现有Excel表头作为字段映射
- **配置记忆**：退出后自动保存配置，下次打开自动恢复
- **可视化界面**：操作简单，实时日志

## 安装

### 方式一：直接运行（需要 Python 3.8+）

```bash
pip install -r requirements.txt
python main.py
```

### 方式二：打包成独立 .exe（不需要安装 Python）

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name "OCR填表工具" main.py
```

打包后在 `dist/OCR填表工具.exe`，可直接发给别人使用。

## 使用步骤

1. **选择图片文件夹** → 点击"扫描图片"
2. **选择/创建 Excel 文件**
   - 已有模板：选"选择已有..."
   - 新文件：先添加字段映射，再点"新建模板"
3. **配置字段映射**
   - 方式A：手动"添加字段"（字段名 = 图片中出现的文字，列 = Excel中的A/B/C...）
   - 方式B：点击"从样例图片识别" → OCR后勾选字段名
   - 方式C：已有Excel → 点击"从Excel表头导入"
4. **点击"开始识别并填表"** → 等待完成

## Excel 格式

| A（姓名） | B（性别） | C（年龄） | D（身份证号） |
|-----------|-----------|-----------|--------------|
| 张三       | 男         | 28        | 410...        |

表头与字段名一一对应，程序会自动将识别的内容填入对应列。

## 图片格式要求

- 支持的格式：jpg/jpeg、png、bmp、tiff、webp
- **图片要清晰**，文字无明显变形、倾斜
- 图片中字段格式建议：`字段名：值` 或 `字段名 值`

## 注意事项

- 首次运行会下载OCR模型（约15MB），稍等片刻
- 大量图片处理时请耐心等待，日志会实时显示进度
- 识别失败会标记为 "（未识别）"，不会漏行
- 每次运行是**追加写入**，不会覆盖已有数据
